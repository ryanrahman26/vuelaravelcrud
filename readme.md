# vuelaravelcrud

## Install:
- git clone https://github.com/ryanrahman26/vuelaravelcrud.git
- cd vuelaravelcrud
- composer install
- npm install
- npm install vue-router vue-axios --save
- npm run dev
- Create DB `vuecrud`, copy .env.example to .env, SET USERNAME, PASSWORD, DATABASE
- php artisan migrate
- php artisan key:generate
- php artisan serve
